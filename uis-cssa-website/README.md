# UiS 华人学生学者联合会网站

这是 UiS 华人学生学者联合会 (CSSA-UiS) 的官方网站。

## 功能特点

- 响应式设计，适配各种设备
- 新生指南信息
- 活动展示
- 关于我们和组织介绍
- 联系方式

## 部署说明

本网站使用 GitHub Pages 免费托管。

访问地址: https://[您的用户名].github.io/uis-cssa

## 本地开发

1. 克隆仓库
2. 在浏览器中打开 `index.html` 文件

## 更新内容

要更新网站内容，只需编辑相应的 HTML、CSS 或 JavaScript 文件，然后提交更改到 GitHub 仓库。
